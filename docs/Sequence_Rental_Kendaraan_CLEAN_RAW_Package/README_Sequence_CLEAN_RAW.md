# Sequence Diagram Rental Kendaraan - Clean RAW

Versi sequence diagram ini dibuat ulang agar lebih rapi:
- Lifeline disusun horizontal dengan jarak besar.
- Pesan disusun dari atas ke bawah secara kronologis.
- Teks pesan tidak ditumpuk dalam satu garis.
- Ada activation box.
- Ada fragment alt untuk stok kosong dan pembayaran.
- Format RAW XML tanpa Base64/compression.

Gunakan file `.drawio` untuk import pertama. Jika gagal, gunakan `.xml`.
